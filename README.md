# the game of life
simple python implementation of [the game of life](https://en.wikipedia.org/wiki/Conway's_Game_of_Life)

## installation
```
$ git clone https://github.com/PanMuminek/the-game-of-life.git
```
## running

### linux/mac
```
$ cd the-game-of-life 
$ python3 view.py
 ```
